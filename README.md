## 📌 Basic Rule

- 정해진 기간 안에 **블로그 포스트를 작성**하고 **메인 README.md 파일에 글 링크를 업데이트**합니다.
- 개인 [닉네임].md파일을 포함하여 모든 업데이트 사항은 직접 편집합니다.
- 해당 주에 작성하지 못한 경우 벌금으로 기록됩니다. 벌금은 매 달 마지막 주에 정산합니다.
